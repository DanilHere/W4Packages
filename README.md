A simple package manager for Linux with python and menu interfaces

## Features

- Update and Upgrade Repository 
- Install Package
- Uninstall Package
- Remove Cache and Junk file
- Supports many Linux Distro (Debian/Ubuntu, Arch, Fedora, and Alpine Linux)
- etc
